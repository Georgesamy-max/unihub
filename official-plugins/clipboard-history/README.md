# 剪贴板历史管理插件

一个功能强大的剪贴板历史管理工具，帮助你记录、搜索和管理剪贴板内容。

## 功能特性

- 📋 **自动记录** - 自动监控并记录剪贴板内容
- 🔍 **智能搜索** - 快速搜索历史记录
- 📌 **固定功能** - 固定重要内容，防止被清除
- 🏷️ **类型识别** - 自动识别文本、链接、代码等类型
- 🎨 **优雅界面** - 使用 Radix Vue 和 Tailwind CSS 构建
- 💾 **本地存储** - 所有数据保存在本地，保护隐私
- 🌓 **深色模式** - 支持亮色和深色主题

## 使用方法

1. 复制任何内容到剪贴板，插件会自动记录
2. 点击任意历史记录可快速复制
3. 使用搜索框查找特定内容
4. 点击固定图标保护重要内容
5. 按类型筛选：文本、链接、代码

## 开发

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build

# 打包
pnpm package
```

## 技术栈

- Vue 3
- Radix Vue (rekaui)
- Tailwind CSS 4
- VueUse
- Lucide Icons
