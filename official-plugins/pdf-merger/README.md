# PDF 合并工具

一个简单易用的 PDF 文件合并工具，支持拖拽排序和批量处理。

## 功能特性

- ✅ 支持多个 PDF 文件合并
- ✅ 拖拽上传文件
- ✅ 拖拽调整合并顺序
- ✅ 显示文件信息（大小、页数）
- ✅ 本地处理，无需上传服务器
- ✅ 支持深色模式

## 使用方法

1. 点击"添加 PDF"按钮或拖拽 PDF 文件到窗口
2. 拖动文件卡片调整合并顺序
3. 点击"合并 PDF"按钮生成合并后的文件
4. 自动下载合并后的 PDF

## 技术栈

- Vue 3
- TypeScript
- pdf-lib
- Tailwind CSS
- Vite

## 开发

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build

# 打包插件
pnpm package
```
